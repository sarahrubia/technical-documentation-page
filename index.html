Nicotine+

Nicotine+ é um cliente gráfico para a rede peer-to-peer Soulseek.

O Nicotine+ pretende ser uma alternativa agradável, gratuita e de código aberto (FOSS) ao cliente oficial Soulseek, fornecendo funcionalidades adicionais e mantendo-se atualizado com o protocolo Soulseek.

Nicotine+ usa GTK para sua interface gráfica de usuário e é escrito em Python.

Download Nicotine+

A versão estável atual do Nicotine+ é 3.2.1, lançada em 10 de fevereiro de 2022. Veja as notas de lançamento.

Downloads estão disponíveis para

  • GNU/Linux, *BSD e Solaris
  • Windows
  • macOS

GNU/Linux, *BSD, Solaris

Ubuntu/Debian

Para usar pacotes estáveis no Ubuntu e no Debian, adicione o repositório PPA nicotine-team/stable executando o seguinte:

sudo apt install software-properties-common
sudo add-apt-repository ppa:nicotine-team/stable
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 6CEB6050A30E5769
sudo apt update
sudo apt install nicotine

Fedora

Para instalar o Nicotine+ no Fedora, execute o seguinte:

sudo dnf install nicotine+

Flatpak

Se sua distribuição Linux suporta Flatpak, você pode instalar a versão estável atual do Nicotine+ do Flathub.

Arch Linux/Manjaro/Parabola

Nicotine+ está disponível no repositório da comunidade do Arch Linux, Manjaro e Parabola. Para instalar, execute o seguinte:

sudo pacman -S nicotine+

Void Linux

Para instalar o Nicotine+ no Void Linux, execute o seguinte:

sudo xbps-install -S nicotine+

FreeBSD

Para instalar o Nicotine+ no FreeBSD, execute o seguinte:

pkg install py-nicotine-plus

pip

Se o Nicotine+ não estiver empacotado para o seu sistema, a versão estável atual pode ser instalada usando o pip. Certifique-se de que as dependências de tempo de execução estejam instaladas e execute o seguinte:

pip3 install nicotine-plus

Tenha em mente que o Nicotine+ não será atualizado automaticamente. Quando uma nova versão estiver disponível, execute o seguinte:

pip3 install --upgrade nicotine-plus

Windows

Instaladores estáveis do Windows para Nicotine+ estão disponíveis para download. A instalação do Nicotine+ requer privilégios de administrador.

NOTA: O formato do instalador mudou desde o Nicotine+ 3.2.0. Se você estiver atualizando do Nicotine+ 3.1.1 ou anterior, desinstale o Nicotine+ primeiro (isso não removerá suas configurações existentes).

Jurídico e Privacidade

  • Ao se conectar ao servidor padrão do Soulseek, você concorda em cumprir as regras e os termos de serviço do Soulseek.
  • Nicotine+ não coleta nenhum dado do usuário. Políticas diferentes podem ser aplicadas ao servidor Soulseek padrão, que não é operado pela Equipe Nicotine+.
